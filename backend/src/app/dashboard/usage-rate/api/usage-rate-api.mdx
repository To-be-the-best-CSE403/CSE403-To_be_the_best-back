# Usage Rate API

## Usage Rate

This page features a high-level view of how the API works for finding the usage rate of a Pokémon.
You can use the drop-down menu to select a Pokémon and use the url request to get the corresponding data.
An example is displayed below.

### API Endpoint

<UsageRateEndpoint />

### Request

Using Javascript:

<CodeHighlight
  language="javascript"
  code={`
const url = "https://tobethebest.vercel.app/api/usage-rate?pokemon=GreatTusk";
fetch(url)
  .then((res) => res.json())
  .then((data) => console.log(data));
`}
/>

### Response

<CodeHighlight
  language="json"
  code={`
{
    "raw_count": 981696,
    "real_count": 753970,
    "usage_rate": 30.13411
}
`}
/>
