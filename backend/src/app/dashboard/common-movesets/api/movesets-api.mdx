# Common Movesets API

This page features a high-level view of how the API works for finding the common moveset of a Pokémon.
You can use the drop-down menu to select a Pokémon and use the url request to get the corresponding data.
An example is displayed below.

### Request

Using Javascript:

<CodeHighlight
  language="javascript"
  code={`
const url = "https://tobethebest.vercel.app/api/movesets?pokemon=greattusk";
fetch(url)
  .then((res) => res.json())
  .then((data) => console.log(data));
`}
/>
