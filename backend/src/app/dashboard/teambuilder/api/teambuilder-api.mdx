# Teambuilder API

This page features a high-level view of how the API works for selecting an team archetype of Pok√©mon.
You can use the drop-down menu to select an archetype and use the url request to get the corresponding data.
An example is displayed below.

## Archetype

- Hyper Offense **ho**
- Stall **stall**
- Bulky Offense **bo**
- Balance **balance**
- Weather **weather**

### API Endpoint

<ArchetypeEndpoint />

### Request

Using Javascript:

<CodeHighlight
  language="javascript"
  code={`
const url = "https://tobethebest.vercel.app/api/teambuilder?archetype=ho";
fetch(url)
  .then((res) => res.json())
  .then((data) => console.log(data));
`}
/>
